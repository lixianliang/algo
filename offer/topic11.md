# 旋转数组的最小数字

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。例如，数组 [3,4,5,1,2] 为 [1,2,3,4,5] 的一个旋转，该数组的最小值为1。  

示例 1：

输入：[3,4,5,1,2]
输出：1
示例 2：

输入：[2,2,2,0,1]
输出：0


## 题解

### 方法1
    循环遍历
``` shell
    时间复杂度为O(N)
    初始化值为arr[0]
    循环遍历数组，若arr[i] < arr[i-1]，则是arr[i]
```

### 方法2
    二分查找
    时间复杂度为O（LOGN）
    如果x >= arr[min]则在前半段子数组
    x <= arr[max] 则在后半段子数组
    
    当min+1 = max就终止条件

    需要考虑arr[min] arr[mid] arr[max]相等的情况
